# Async
* 비동기 처리 : 실행 결과를 기다리지 않고 다른 작업을 할 수 있도록 처리하는 방식
* 대기 시간이 긴 작업을 처리할 때 유용하다.

* 자바에서 멀티스레딩을 사용해 비동기 처리를 구현할 수 있다.
